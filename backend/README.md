# Hapi-Boilerplate

Little boilerplate for Hapi-Api-Apps.

## Install

`HBP=PROJECTNAME && git clone https://github.com/platdesign/hapi-boilerplate.git $HBP && cd $HBP && npm i && rm -rf .git`

## Commands

```sh
# Run test
npm test

# Start in dev-mode
npm run dev

# Start in production mode
npm start
```
